# Email Signature Setup Instructions

I've created two professional email signature templates for Praveen & Partners:

## Files Created

1. **email-signature.html** - Full-featured signature with CTA button
2. **email-signature-plain.html** - Minimal version (better compatibility)

## How to Install

### For Gmail:

1. Open the HTML file in your browser (double-click `email-signature-plain.html`)
2. Select all content (Cmd+A / Ctrl+A)
3. Copy (Cmd+C / Ctrl+C)
4. Go to Gmail Settings → See all settings → General
5. Scroll to "Signature" section
6. Click "Create new" and give it a name
7. Paste (Cmd+V / Ctrl+V) into the signature box
8. Scroll down and click "Save Changes"

### For Outlook (Mac):

1. Open the HTML file in your browser
2. Select all content (Cmd+A)
3. Copy (Cmd+C)
4. Open Outlook → Preferences → Signatures
5. Click "+" to create new signature
6. Paste (Cmd+V) into the signature area
7. Set as default for new messages

### For Outlook (Windows):

1. Open the HTML file in your browser
2. Select all content (Ctrl+A)
3. Copy (Ctrl+C)
4. Open Outlook → File → Options → Mail → Signatures
5. Click "New" to create new signature
6. Paste (Ctrl+V) into the signature editor
7. Click "OK" to save

### For Apple Mail:

1. Open the HTML file in your browser
2. Select all content (Cmd+A)
3. Copy (Cmd+C)
4. Open Mail → Preferences → Signatures
5. Select your email account
6. Click "+" to add new signature
7. Paste (Cmd+V) into the signature area
8. Close preferences (auto-saves)

## Customization

To customize your signature, edit the HTML files:

- **Name**: Change "Praveen" to your name
- **Title**: Change "Founder & Technical Director" to your role
- **Email**: Update `praveen@praveenpartners.com`
